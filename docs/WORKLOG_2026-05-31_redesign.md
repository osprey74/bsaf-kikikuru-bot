#### XML 構造の核心

```text
Report
├ Control (Title, Status, EditorialOffice, DateTime)
├ Head/Headline (要約 — 4 階層 Information)
└ Body
    ├ Warning[@type="気象警報・注意報（府県予報区等）"]    ← Item × N
    ├ Warning[@type="気象警報・注意報（一次細分区域等）"]  ← Item × N
    ├ Warning[@type="気象警報・注意報（市町村等をまとめた地域等）"]
    ├ Warning[@type="気象警報・注意報（市町村等）"]        ← ★ 量予想・警戒レベルの主データ
    │  └ Item
    │      └ Kind[]（VPWW61 では複数並列出現あり）
    │          ├ Name, Code, Status, LastKind, Addition
    │          └ Property[]
    │              ├ Type ("大雨浸水危険度"/"風危険度" 等)
    │              ├ SignificancyPart/Base/Significancy/Code ← ★警戒レベル
    │              └ WindSpeedPart, WaveHeightPart, ... (量予想)
    ├ Warning[@type="気象警報・注意報（高潮予報区間）"]   ← VPWW57 のみ 5 階層目
    └ AdditionalInfo (VPWW57 のみ)
```
